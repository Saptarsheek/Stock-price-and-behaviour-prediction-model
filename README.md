# Stock-market-price-and-behaviour-prediction-model
This project focuses on predicting stock market prices and analyzing market behavior using machine learning techniques. The model uses historical stock data such as open, high, low, close prices, and trading volume to identify patterns and trends.

Machine learning algorithms like regression and boosting models (e.g., XGBoost/LightGBM) are applied to learn from past data, enabling the prediction of future price movements and classification of market behavior (bullish or bearish trends).

The project is divided into two main parts:

Model Development (Jupyter Notebook): Includes data preprocessing, feature engineering, model training, and evaluation.
Web Application (Streamlit in VS Code): A user-friendly interface where users can upload data and get predictions in real-time, deployed as a working hostable dashboard.

The goal of this project is to support data-driven decision-making by providing insights into potential stock price directions and market dynamics.
